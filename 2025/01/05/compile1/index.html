编译原理八股文——文法和语言

文法和语言

符号、符号串及其集合的运算

任何一种语言都是由该语言的基本符号组成的符号串的集合

字母表和符号串

字母表是元素的非空又穷集合，又称符号集，例如对于全部由小写字母组成的字符串，其字母表为 {a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z}

符号串是字母表中的符号组成的任何有序序列

字母表上的运算

  • 字母表的乘积：{a, b}{0, 1} = {a0, a1, b0, b1}
  • 字母表的 n 次幂：{0, 1}3 = {0, 1}{0, 1}{0, 1}，0 次幂是空串
  • 字母表的正闭包：字母表从 1 次方到 n 次方求并
  • 字母表的克林闭包：正闭包加空串

串

串是字母表中符号的一个有穷序列，串的长度指串中符号个数，空串长度为 0

文法的直观概念

对语言结构的描述和定义，即在形式上用来描述和规定语言结构的称为 “文法”

规则

<句子> -> <名词短语><动词短语>
<名词短语> -> <形容词><名词短语>
<形容词> -> little
…
尖括号中是语法成分，未用尖括号的部分是语言的基本符号

每一条都是一个规则，也称为产生式

推导

使用某一条规则，用左半部分替换右半部分，称为使用规则推导句子

文法和语言的形式化定义

  • VT：终结符集合
  • VN：非终结符集合
  • P：产生式集合
  • S：开始符号

推导和规约

句子的推导是从开始符号起，不断用表达式的左部去替换右部，直到推导出特定句子

句子的归约是从指定句子起，不断将句型的成分根据表达式变成左部，直到归约出开始符号

  • 句子的推导：从生成语言的角度
  • 句子的归约：从识别语言的角度

语言的形式化定义

由文法 G 开始符号 S 推导出的所有句子构成的集合称为文法 G 生成的语言，记为 L(G)

  • 句型：推导/归约过程中既可以包含终结符也可以包含非终结符的部分
  • 句子：推导/归约过程中只包含终结符的句型

两个文法生成的语言相同，即 L(G1) = L(G2)，则称 G1 和 G2 等价

文法的类型

0 型文法 —— 无限制文法

1 型文法 —— 上下文有关文法 CSG

2 型文法 —— 上下文无关文法 CFG

3 型文法 —— 正则文法 RG

CFG 语法分析树

根节点的标号为文法开始符号

内部结点表示一个产生式的应用，该结点的标号是此产生式左部。该结点的子结点的标号从左到右构成了产生式的右部

叶节点的标号既可以是终结符也可以是非终结符，从左到右排列叶节点得到的符号串称为这棵树的产出或边缘

每一个推导对应一棵分析树

短语、直接短语、句柄

给定一个句型，其分析树中的每一棵子树的边缘称为该句型的一个短语

如果子树只有父子两代结点，那么这棵子树的边缘称为该句型的一个直接短语

对于一棵语法树，所有非叶子结点构成的子树的边缘的集合为短语集合，如果子树只有两层，则为直接短语

最靠左的一棵只有两层的子树对应的直接短语称为句柄

二义性

如果一个文法可以为某个句子生成多棵分析树，则称这个文法是二义的

我们无法检验一个文法是否具有二义性，但是可以提出一组规则，满足这组规则的文法一定没有二义性

有关文法实际应用的一些说明

文法中不能含有有害规则和多余规则

  • 有害规则：例如 D -> D 这类会引起二义性的推导
  • 多余规则：文法中任何句子的推导都不会用到的规则，不可到达 or 不可终止
