miniob 框架解析

我负责的题目

drop table

题目要求删除某个表格，一个表格的信息包括三种：数据信息 + 索引信息 + 表格元数据

  1. 数据信息：真实存放到表格当中的数据
  2. 索引信息：一个索引对应一棵 B+ 树
  3. 表格元数据：各种表格的描述信息

drop table 不需要算子，需要执行器
