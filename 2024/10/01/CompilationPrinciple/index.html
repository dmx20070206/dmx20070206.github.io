文法和语言的形式定义

文法 定位为四元组 : 非终结符集合 : 终结符集合 : 规则 : 开始符/起始符

对于标识符来说(字母打头的字母数字序列)： 标识符文法 : (1) = {标识符, 字母, 数字} (2) = {a,b,c,...,x,y,z,...,0,1,...,9} (3) = { <标识符> -> <字母> <标识符> -> <标识符><字母> <标识符> -> <标识符><数字> <字母> -> a <字母> -> b ... <字母> -> z <数字> -> 0 <数字> -> 1 ... <数字> -> 9 } (4) = <标识符>

也即字母表 也即两者不相交

对于一个文法 ，从识别串推到而来的符号串称为句型，只包含终结符的句型称为句子，全部句子的集合称为文法 描述的语言。如果两个文法描述的语言是一致的，则称这两个文法是等价的

文法的类型

0型文法 每个产生式 中， 至少包含一个非终结符 1型文法/上下文有关文法 每个产生式 中， 2型文法/上下文无关文法 每个产生式 中， 是一个非终结符 3型文法/正规文法 每个产生式 中，都是 的形式

四种文法是逐级增加限制的关系，也是逐级包含的关系
