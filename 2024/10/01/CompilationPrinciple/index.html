编译原理课内知识笔记

本文章对标清华大学紫皮书《编译原理》

第二章 文法和语言

文法，是语言的规则

符号与符号串

字母表，是元素的非空有穷集合，一般用 $\displaystyle\sum$ 表示
符号串，是字母表中符号的任何有穷序列，空符号串表示为 $\varepsilon$

符号串的前缀和后缀称为符号串的头和尾，不包含符号串本身的头和尾也称作固有头和固有尾

字母表的闭包：
$\displaystyle\sum^{*} = \displaystyle\sum^{0} \cup \displaystyle\sum^{1} \cup \dots \cup \displaystyle\sum^{n} \dots$
字母表的正闭包：
$\displaystyle\sum^{+} = \displaystyle\sum^{1} \cup \displaystyle\sum^{2} \cup \dots \cup \displaystyle\sum^{n} \dots$

文法和语言的形式定义

文法 $G$ 定位为四元组 $G=(V_{N},V_{T},P,S)$
$V_{N}$ : 非终结符集合
$V_{T}$ : 终结符集合
$P$ : 规则
$S$ : 开始符/起始符

对于标识符来说(字母打头的字母数字序列)：
标识符文法 $G$ :
(1) $V_{N}$ = {标识符, 字母, 数字}
(2) $V_{T}$ = {a,b,c,…,x,y,z,…,0,1,…,9}
(3) $P$ = {
<标识符> -> <字母>
<标识符> -> <标识符><字母>
<标识符> -> <标识符><数字>
<字母> -> a
<字母> -> b
…
<字母> -> z
<数字> -> 0
<数字> -> 1
…
<数字> -> 9
}
(4) $S$ = <标识符>

$V_{T} \cup V_{N} = V$ 也即字母表
$V_{T} \cap V_{N} = \emptyset$ 也即两者不相交

若符号串 $v$ 能通过一条规则变成符号串 $w$，则称 $w$ 是 $v$ 的直接推导，记作 $v \Rightarrow w$
如果有这样一条推导链 $v \Rightarrow w_{0} \Rightarrow w_{1} \Rightarrow w_{2} \Rightarrow \dots \Rightarrow w_{n} = w$，则称 $v \Rightarrow ^{+} w$

对于一个文法 $G$，从识别串推到而来的符号串称为句型，只包含终结符的句型称为句子，全部句子的集合称为文法 $G$ 描述的语言。如果两个文法描述的语言是一致的，则称这两个文法是等价的

文法的类型

0型文法
每个产生式 $\alpha \rightarrow \beta$ 中，$\alpha$ 至少包含一个非终结符
1型文法/上下文有关文法
每个产生式 $\alpha \rightarrow \beta$ 中，$|\beta| \geq |\alpha|$
2型文法/上下文无关文法
每个产生式 $\alpha \rightarrow \beta$ 中，$\alpha$ 是一个非终结符
3型文法/正规文法
每个产生式 $\alpha \rightarrow \beta$ 中，都是 $\alpha \rightarrow a or \alpha \rightarrow a\beta$ 的形式

四种文法是逐级增加限制的关系，也是逐级包含的关系

对于文法 $G=({S,A},{a,b},P,S)$
$$
P = {
S \rightarrow aAS
A \rightarrow SbA
A \rightarrow SS
S \rightarrow a
A \rightarrow ba
}
$$ 的其中一颗语法树如下图：

该语法树对应的句子为 $aabbaa$

如果从标识符推导到句子的过程中，都是对句型的最左(最右)非终结符进行替换，则称这种推导为最左(最右)推导，最右推导也称为规范推导

若某个文法存在某个句子对应两棵不同的语法树，则称这个文法是二义的
实际应用中，我们需要寻找一组充分条件，尽可能通过额外的限制来消除文法的二义性
